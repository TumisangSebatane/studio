# **App Name**: LexiBotswana PWA

## Core Features:

- Offline Document Storage: Enable offline access to legal documents by storing them locally within the PWA.
- Responsive Document Display: Display legal texts in a clean, responsive interface.
- Text Highlighting: Highlight and annotate legal documents.
- Instant Legal Q&A: Enable fast queries with the tool powered by Phi-3 Mini, to instantly answer legal questions.
- Local Profile Persistence: Allow saving and retrieving user-specific preferences locally.

## Style Guidelines:

- Primary color: Deep, authoritative blue (#30475E), reflecting the legal field.
- Background color: Light gray (#D6D5D6) for a professional and readable environment.
- Accent color: Soft gold (#E0A96F) to highlight important features and CTAs, referencing Botswana's natural resources.
- Body: 'PT Sans' (sans-serif) for readability and modernity; Headline: 'Literata' (serif) for titles and emphasized text.
- Code font: 'Source Code Pro' for any code snippets that appear (though none are currently planned), such as showing JSON examples.
- Clean, tab-based navigation to organize document access, search, and user settings, suitable for both mobile and desktop.
- Use formal, outline-style icons to indicate legal functions, in gold or the primary blue.
